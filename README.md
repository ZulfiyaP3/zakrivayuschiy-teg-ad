# Закрывающий тег

## Описание проекта

"Закрывающий тег" — это веб-сайт с пиксельным дизайном, выполненный в стиле ретро-игр. Проект представляет собой блог-платформу для публикации воспоминаний о процессе обучения веб-разработке.

### Основные возможности:

- **Адаптивный дизайн** — корректное отображение на мобильных устройствах и десктопах
- **Интерактивные кнопки лайка** с анимированным сердцем:
  - Исходное состояние: белое сердце с черным контуром
  - Ховер-эффект: черное сердце при наведении
  - Активное состояние: красное сердце при нажатии
  - Лайкнутое состояние: красное сердце с текстом "Unlike"
- **Модальное окно** для сохранения постов
- **Покадровая анимация** заполнения сердца при взаимодействии
- **Пиксельная графика** и ретро-стилистика

### Технологии:

- HTML5
- CSS3 (Flexbox, Grid, анимации, медиа-запросы)
- JavaScript (ES6+)
- SVG для иконок и графики
- Кастомные шрифты (PressStart2P, Inter-Variable)

## Инструкция по использованию

### Системные требования:

- Современный веб-браузер (Chrome, Firefox, Safari, Edge последних версий)
- Локальный веб-сервер (опционально, для корректной работы некоторых функций)

### Запуск проекта:

1. Клонируйте репозиторий или распакуйте архив проекта
2. Откройте файл `index.html` в браузере

   Или используйте локальный веб-сервер:

   ```bash
   # Python 3
   python3 -m http.server 8000
   
   # Node.js (если установлен http-server)
   npx http-server
   ```

3. Откройте в браузере: `http://localhost:8000`

### Использование:

- **Лайки**: Наведите курсор на кнопку "Like" — сердце станет черным. Кликните — сердце станет красным, текст изменится на "Unlike"
- **Сохранение**: Нажмите кнопку "Сохранить на память" в футере для открытия модального окна
